# DevDrop

DevDrop is a local-first "Dropbox for developers" CLI prototype.

The MVP will prove this workflow:

1. Initialize a workspace.
2. Scan project structure and Git metadata.
3. Generate a canonical manifest.
4. Recreate safe placeholder structure.
5. Hydrate Git repositories on demand.
6. Manage encrypted environment profiles without printing secret values.

Implementation is tracked in Linear project `DevDrop MVP` and prepared for the
`wave-ship` workflow using `ops/wave-ship/devdrop-mvp.args.json`.
